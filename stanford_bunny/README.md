# ComputerGraphics

## Stanford bunny

#### How to run

Clone the repository<br/> and checkout on the 'stanford_bunny' branch:

```
git clone https://github.com/lergor/ComputerGraphics.git
cd ComputerGraphics
git checkout stanford_bunny
```

In the 'stanford_bunny' directory run:
```
mkdir build
cd build
cmake .. && make
./stanford_bunny
```
